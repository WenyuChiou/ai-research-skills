## Style guide

- Soft blue / white background, one accent colour (orange) for human-owned stages (3a, 4) and the "Gap" callout (stage 6).
- Skill names rendered as monospace pill chips so they read as code identifiers.
- Cross-cutting tools shown in a separate band with dotted lines back to the stages they support.
- Repo URL in the bottom-right legend.
