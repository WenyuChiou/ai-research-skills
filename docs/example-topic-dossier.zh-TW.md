## 2. 候選主題定義

**候選主題 1 — LLM 驅動的水資源管理。** 使用 LLM 與 LLM 代理人進行水資源管理和決策支援：水文預測、水庫與河流操作、監測及規劃。*為何可能是缺口：* 未曾有人嘗試過 — 將 LLM 此一新近能力廣泛應用於水資源管理。

**候選主題 2 — 用於社會水文學中人類行為的 LLM 代理人。** 在一個人與水耦合（社會水文學）模型中，使用 LLM 代理人來代表個別利害關係人和家戶如何做出用水決策，並根據觀察到的用水和調適行為對這些代理人進行校準與檢驗。*為何可能是缺口：* 未曾有人嘗試過 — 尚無為社會水文學模型中利害關係人行為建立的校準過的 LLM 代理人。

**最接近的先期研究。**

- **候選主題 1：** 此領域已被廣泛研究。關鍵項目包括：一個水文學專屬的 LLM 基準資料集 (Kizilkaya et al. 2025, *HydroLLM* — 原創研究)；一篇關於 LLM 在水文學中應用的文獻計量分析 (Sajja et al. 2025 — 綜述)；一篇關於 AI 在水資源規範中應用的系統性綜述 (Wang et al. 2026 — 綜述)；為環境問題微調的 LLM (Zhang et al. 2025 — 原創研究)；以及對話式水質 LLM 代理人 (Ravindran et al. 2025 — 原創研究)。一個基準加上一篇文獻計量研究是明確的領域成熟度信號 — 已被佔據的信號相當穩固。
- **候選主題 2：** 尚無論文為社會水文學建立校準過的 LLM 代理人。領域內最接近的研究是 Batista et al. 2025 (相近類比 — 一個為水資源治理模型進行*情感*分析的 LLM，而非基於代理人的決策模型)。 Schück (2026) 是主要的警示論文。Braga et al. 2025 和 Khaki et al. 2025 是領域外的 LLM 人類行為類比研究。直接證據薄弱 — 這正是缺口保持開放的原因。

## 5. 逐關卡評估

### 關卡 1 — 缺口仍開放

- **分數。** 候選主題 1：1/5 (強烈不同意)。候選主題 2：3/5 (中立)。
- **證據。** 對於候選主題 1，此領域已有一個基準、一篇系統性綜述、一篇文獻計量分析，以及原創的微調和對話式代理人研究 — 廣泛的缺口已被確實佔據。對於候選主題 2，尚無論文完成校準過的LLM代理人於社會水文學之任務；只有一篇領域內的相近類比 (Batista 2025, 情感分析) 和一篇警示論文 (Schück 2026) 與此議題相關。
- **詮釋。** 候選主題 1 已被佔據；維持現述推進將產出該領域已有的研究。候選主題 2 看似開放，但中等信心的檢索意味著此結論為暫時性的。
- **風險。** 執行期間 Semantic Scholar 受到速率限制，因此可能遺漏了與候選主題 2 相關的論文。「缺席於此文獻庫」不等於「開放」的證明。
- **所需行動。** 在依賴候選主題 2 的開放性之前，需啟用 Semantic Scholar API 金鑰重新執行檢索。

### 關卡 2 — 實質貢獻

- **分數。** 候選主題 1：未評估 (未通過關卡 1)。候選主題 2：3/5 (中立)。
- **證據。** 未發現已放棄的嘗試。Schück (2026) 是一篇經同儕審查的觀點論文，其前提正是一個持續存在的警示。Batista (2025) 是領域內最接近的先期研究，使用 LLM 於水資源治理的人類信號，但其為情感分析，而非基於代理人的決策模型。
- **詮釋。** 候選主題 2 將貢獻的是**一種用 LLM 代理人來代表家戶用水調適決策的已驗證方法** — 這些代理人的選擇經校準，並與觀察到的調適行為進行比對測試，可作為社會水文學模型的行為層。其貢獻在於已驗證的行為表徵，而非「LLM 代理人應用於水資源」。創新性屬界線模糊；驗證要求是能將此研究從延伸性工作提升為新能力的關鍵。
- **風險。** **構念效度** — LLM 生成的行為是否忠實地代表真實的人類用水決策，或僅是看似合理。這正是 Schück 提出的風險。
- **所需行動。** 提案必須滿足的最低驗證草圖：行為目標 = 觀察到的家戶調適決策（例如，家戶是否採取防洪措施或在缺水時改變用水行為）；基線 = 傳統的離散選擇模型或規則式 ABM 代理人；留出測試 = 在 LLM 代理人未經調校的留出家戶集合上預測決策；研究必須克服的主要失敗模式 = 看似合理但與母體不符的行為。

### 關卡 3 — 可行性

- **分數。** 候選主題 1：未評估 (未通過關卡 1)。候選主題 2：3/5 (中立)。
- **證據。** 家戶用水與調適行為資料部分公開；水文資料大多公開；LLM API 存取權可以計量費用取得。基線模型 (離散選擇模型 / 規則式 ABM) 是標準的。家戶行為資料涉及隱私與同意限制 — 目標性調查需倫理審查批准。再現性是一項隱憂：LLM 輸出依賴外部 API 模型。一篇碩士論文規模的專案，適合一個區域、數百家戶的行為資料、一個社會水文學耦合，大約需 12–18 個月的工作；LLM API 成本適中 (數百美元)。
- **詮釋。** **主要限制**是行為驗證資料集。目前提案可行 — 設計可被完整闡述。論文可行性取決於資料集：若存在可重複使用的行為資料集，則一篇完整的論文是實際的；若需初級資料蒐集，專案仍可行，但資料蒐集階段將佔據第一年的主要時間。
- **風險。** 家戶行為資料的隱私與同意限制，加上再現性 — LLM 代理人輸出依賴可能改變或淘汰的外部 API 模型。
- **所需行動。** 找出一可重複使用的觀察行為資料集 (一個值得先查核的線索：Jiang et al. 2026 為華北平原建立了包含政策情境的社會水文學模型)。若無，則規劃一個符合專案時程並包含倫理審查批准的初級資料蒐集計畫。鎖定 LLM 模型版本並存檔提示與輸出以確保再現性。

## 6. 風險與升級／否決測試

**已識別風險。**

- **構念效度風險** — LLM 生成的行為是否忠實地代表真實的人類用水決策，或僅是看似合理；這正是 Schück (2026) 針對候選主題 2 所採方法提出的風險。
- **資料集限制風險** — 行為驗證資料集是候選主題 2 可行性的主要限制；若找不到可重複使用的資料集，初級資料蒐集將佔據主要時程。
- **創新性風險** — 候選主題 2 介於新能力與延伸性工作之間的界線；Batista 2025 是一個較弱的已實現形式 (用於水資源治理的 LLM 情感分析)，專案必須超越之。
- **再現性風險** — LLM 輸出依賴外部 API 模型；設計必須鎖定模型版本並存檔提示與輸出，以便在 API 模型改變或淘汰時能重新執行研究。

**升級／否決測試 — 候選主題 2：用於社會水文學中人類行為的 LLM 代理人。** 當**所有**以下條件成立時，值得推進：

1. 啟用 Semantic Scholar 的全召回檢索重新執行後，未發現任何已建立用於社會水文學的校準過 LLM 代理人之論文。
2. 一項先導研究產出留出測試驗證結果，其中 LLM 代理人預測觀察到的家戶調適決策的表現，至少與在相同留出集合上的離散選擇或規則式 ABM 基線相當。
3. 已找到一可重複使用的觀察行為資料集，或已備妥一份符合專案時程並包含倫理審查批准的初級資料蒐集計畫。

若重新執行檢索後發現此類論文，**或**先導研究顯示代理人無法勝過簡單的基線或無法校準，**或**在範圍內無法取得行為資料集，則不值得推進。

**補救路徑 — 候選主題 1：LLM 驅動的水資源管理。**
此廣泛形式已關閉，但某個特定的服務不足的子問題可能並非如此 — 例如，現有基準和綜述文獻未涵蓋的特定水資源管理任務。要挽救此主題，需要針對該子問題而非廣泛領域進行全新的、更窄的檢索。

## 7. 建議的下一步

不應以目前形式推進廣泛的「LLM 驅動的水資源管理」主題。此領域已穩固建立 — 一個基準、一篇系統性綜述和一篇文獻計量分析均已存在 — 承諾此廣泛框架不會產生具防禦性的貢獻。若對現有文獻服務不足的特定水資源管理子問題感興趣，下一步應是針對該子問題進行全新的、更窄的檢索。

較窄的「用於社會水文學中人類行為的 LLM 代理人」主題仍有條件地看好。在投入之前，需採取三項行動：啟用 Semantic Scholar 重新執行文獻檢索以確認缺口；閱讀 Schück (2026) 並圍繞其構念效度警示設計驗證 — 至少包含對真實家戶調適決策，與離散選擇或規則式 ABM 基線進行留出測試比較；以及，找到一可重複使用的行為資料集，或規劃一個符合專案時程並包含倫理審查批准的初級資料蒐集計畫。
